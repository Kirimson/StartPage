Yo, settings are sort of simple now. Follow the rules to create your own custom StartPage

1. Use the "\" prefix to create a title, then the name of the title. such as: "\Media"
2. After the title on a new line add the link name on one line, and the url on the next
3. Repeat until you've put all links you want
4. Repeat from 1 until you've added all sections. Last line has to be a single "\"
Note: These settings are stored in a cookie. clicking "Create" means you accept all this cookie nonsene.
Cookies are only used for personalisation, nothing more, nothing less. I'm not that fancy
(Also, cookies can only store 4096 bytes (4kb). Using link shorterners may be a good idea if you have a lot of links like I do)
