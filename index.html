Help and Tricks
Advanced search box tricks. Using a search site term, you can instantly search certain sites

- img: - Google Images search
- maps: - Google maps search
- yt: - Youtube search
- ama: - Amazon search
- stack: - Stackoverflow search
- ddg: - Duckduckgo search

Note: Duckduckgo !bangs work when searching with ddg: Learn more about !bangs here

Example: img: cute cats
Google image searches for cute cats
